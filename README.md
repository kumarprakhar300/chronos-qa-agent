# Chronos: Visual Self-Healing QA & Testing Agent

Chronos is an autonomous, VLM-powered E2E QA testing agent that automatically executes test instructions, heals selectors, and provides an interactive visual dashboard.

## Key Features
- **VLM Element Mapping:** Locates target elements visually using screenshots.
- **Self-Healing Loop:** Automatically updates broken or outdated CSS selectors.
- **Live Observability Dashboard:** View the browser execution log, reasoning traces, and visual diffs.
- **Network Interception:** Mock APIs on the fly to test error states.

## Tech Stack
- **Backend:** Node.js, TypeScript, Express, Playwright, Gemini API / VLM.
- **Frontend:** Vite, TypeScript, Vanilla CSS (Premium Glassmorphic Design).
